Endless 100 Trinity

$219.00

Endless 100 Trinity Frames. Those skating trinity mount skates can now enjoy Endless Frame. The Trinity 100 accommodates both 4x100mm AND 3x110mm wheel setups.

For use with 'Bullet Profile' wheels only (small footprint). The 4x100 setup rides rockered while the 3x110 rides flat. Compatible with any trinity mount boot from Powerslide, but it is worth noting that wheel clearance is small if skating 3x110 on Powerslide's fitness skates (i.e. Phuzion models). There is more clearance on the urban boots.


Model Year 2021
Skate Type Urban, Freestyle
Mount Type Trinity
Weight 313g
Boot Material Hybrid Composite
Frame Material CNC Machined 6061-T6 Aluminum
Wheel Size 4x100mm, 3x110mm
